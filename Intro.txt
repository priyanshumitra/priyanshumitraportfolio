HTML Use for structure/layout of website

HTML stands for "Hypertext Markup Language"

HTML is the code that is used to structure a web page and it's content. The component used to Design the structure of
websites are called HTML tags. The extension of  an HTML file is .html or (.htm in rare cases mostly use of extension is .html)

First HTML File must be have name as "index.html"(Home page) Because it's create Home page of our websites. It is the default name
for a website's homepage.

Basic HTML Page Making tags:-
<!DOCTYPE html>       //Defines document type (html5)(Tells browser you are using HTML,It takes our Whole code as HTML5 and convert into web page)
<html>         //Root element of an html page, It is opening part of html code. Every Html page has start from this tag.
   <head>      //container for metadata. The (<head>(open the head part)) element is not visible on the webpage but contains information about the document.

   <body>    //contains all the data renderd by browser. The (<body>(open the body part)) element is visible on the webpage and display it all the information or code under in (<body>) tag.
   <h1>First Programe</h1>  //(<h1>(open the heading)) and (</h1>(close the heading)) use for heading or big characters line
   <p>Hello World!</p>  //(<p>(open the paragraph)) and (</p>(close the paragraph)) use for paragraph or small characters line
   </body>   //contains all the data renderd by browser. The (</body>(close the body part)) element is visible on the webpage and display it all the information or code under in (<body>) tag.
</html>     //Root element of an html page, It is closing part of html code. Every Html page has end by this tag.

Tables in HTML:-

2. <td>: It is used to define data into the cells of a table(used to display table data). 

3. <th>: It is used to define the header of a table(used to display table headers). 

4.<caption> :It is used to give caption on the top of the table for what's the table wants to describe the data about it(Used to display title of table).

5.<thead> : Used to group the header content in a table.(The <thead> element represents the heading information related to the data).

6.<tbody> : This element is not necessary but it is used to group rows belonging to a table body. 

7. <table>: It is used to create an HTML table.(Used to create table). And it is the container of these all tags(<tr>,<td>,<th> and <caption>). these tags are work in <table> tag.

video tag:- 
You can add any videos only from your own device to your own website using the video tag through video link.
Example:- <video src="erwtdh.mp4" height="200" width="200" controls >My Video</video> 
          (controls will show the control buttons on your video like pouse,volume etc.., "erwtdh.mp4" that video already saved in our device so we linked it with our website)

          There are some attributes that we used in video tag:-
          1. control (show the control buttons on your video like pouse,volume etc..)
          2. height  (set the height of your video)
          3. width   (set the width of your video)
          4. autoplay (play the video automatically when user enter in the browser)
          5. loop      (when the video is over it will again start from the first frame)
          6. muted     (It turns off the sound of the video)
